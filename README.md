## Overview
VYass AI is a lightweight Flutter application that showcases a modular, Riverpod‑based architecture for building AI‑enhanced productivity tools. The MVP focuses on a chat interface powered by a mock AI provider, with a scalable codebase ready for real AI integrations (NVIDIA, Gemini, etc.) and persistent storage via Drift (SQLite).

## Key Features
- **Riverpod** state‑management for clean, testable logic.
- **Drift** SQLite with optional encryption support.
- **MockAIProvider** simulating AI responses (`AI: <prompt>`). Real providers can be swapped in seamlessly.
- Cross‑platform UI (Android, iOS, Web, macOS, Linux, Windows) generated by Flutter.
- Comprehensive theming and a splash screen with animated logo.
- Automated testing (unit and widget tests).

## Getting Started

### Prerequisites
- **Flutter SDK** (stable channel) – https://flutter.dev/docs/get‑started/install
- Android SDK and an Android device or emulator (or any other Flutter‑supported platform)
- Git (optional, for cloning)

### Setup & Configuration

1. **Clone the repository (if you haven't already)**
   ```bash
   git clone https://github.com/pheonix6619/Vyaas-AI.git
   cd Vyaas-AI
   ```

2. **Install Flutter dependencies**
   ```bash
   flutter pub get
   ```

3. **Generate Drift code (required for database)**
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```

### API Integration

To enable real AI providers, obtain API keys and add them to the app’s configuration:

- **Gemini (Google AI)**
  - Sign up at the [Google AI Console](https://aistudio.google.com/app/apikey) and create an API key.
  - Add the key to `lib/providers/gemini_provider.dart` (replace the placeholder `YOUR_GEMINI_API_KEY`).

- **NVIDIA**
  - Register at the [NVIDIA AI Playground](https://developer.nvidia.com/ai) and generate an API key.
  - Insert the key into `lib/providers/nvidia_provider.dart` (replace `YOUR_NVIDIA_API_KEY`).

After saving the keys, the app will automatically use the selected provider (see the Settings screen to switch providers).

### Running the App
```bash
# Run on a connected device or emulator
flutter run
```
The app launches with a clean chat screen; type a message and watch the mock AI respond after a short delay.

### Building Release Binaries
```bash
# Android APK (requires a keystore)
flutter build apk --release

# Web
flutter build web

# macOS, Linux, Windows
flutter build <platform>
```
Generated artifacts are placed in `build/`.
